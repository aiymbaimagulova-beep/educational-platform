Курс: HTML & CSS

Этот курс поможет тебе сделать первые шаги в создании сайтов и понять, как устроена веб-страница изнутри.

О чём этот курс

В курсе HTML & CSS ты познакомишься с основами вёрстки, научишься создавать структуру сайта, работать со стилями, цветами и шрифтами. Все темы объясняются простым языком и закрепляются практикой.

Что ты изучишь

HTML-теги, структуру страницы, стили CSS

Формат

Короткие уроки + практические задания

Уровень

Для абсолютных новичков

Что дальше?

После прохождения курса ты сможешь создать свой первый сайт, оформить портфолио и перейти к изучению JavaScript или Python. Это отличный фундамент для старта в IT.
